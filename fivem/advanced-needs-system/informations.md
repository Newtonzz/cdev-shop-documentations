# Informations;



### Default Stats

The script includes the following stats by default:

* **health**
* **armor**
* **breath**
* **hunger**
* **thirst**
* **hygiene**
* **bladder**
* **poop**
* **energy**
* **stress**
* **social**



**Locales**

Translations can be modified or added in:

```
/public/config/locale
```
