# How to Register a New Category!

***

Category Example

```lua
{
    id = "animations",  -- Unique category identifier
    label = "Animations",  -- Display name for the category
}
```

#### Required Fields

* **id**: `string`\
  The unique identifier for the category.
* **label**: `string`\
  A user-friendly name for the category.
